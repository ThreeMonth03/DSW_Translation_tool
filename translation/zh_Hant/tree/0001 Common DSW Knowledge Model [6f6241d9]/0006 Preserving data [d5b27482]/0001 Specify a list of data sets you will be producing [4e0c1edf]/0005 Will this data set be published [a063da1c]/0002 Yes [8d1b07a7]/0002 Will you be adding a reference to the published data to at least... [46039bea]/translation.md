## text

### Source (en)

~~~text
Data is sometimes difficult to locate, especially if it is not in a domain-specific repository. Data catalogues may increase findability: they put the metadata for data sets that are located in different places together in one location that is well known to researchers in a particular field. 

Data sets can be mentioned in multiple data catalogues without the need for duplicating the data: the data catalogues point to where the actual data can be found.

An example is [Narcis](https://dans.knaw.nl/nl/over/diensten/narcis) (for research data in the Netherlands).
~~~

### Translation (zh_Hant)

~~~text
資料有時不好找到，尤其是當資料未存放於領域專門的儲存庫時。資料目錄（data catalogues）有助於提升可查找性（findability）：資料目錄將分散於不同位置的資料集後設資料集中於同一個、且該領域研究人員熟悉的平台。

資料集可以同時被收錄於多個資料目錄，無需重複儲存資料本身：資料目錄可提供指引，連結至實際存放資料的位置。

一個例子是 [Narcis](https://dans.knaw.nl/nl/over/diensten/narcis)（荷蘭的研究資料目錄）。
~~~
