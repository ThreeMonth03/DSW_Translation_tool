## text

### Source (en)

~~~text
Consider that users of a data set will need to be able to get access several years after the project is finished. The best way to arrange this is to put the responsibility for checking the rights to access the data with a fixed body in an organisation, a data access committee, rather than one of the investigators of the project.
~~~

### Translation (zh_Hant)

~~~text
請考量資料集的使用者可能需要在專案結束數年後仍能存取資料。為了妥善安排這一點，最好的做法是將資料取用權限的審核責任，交由組織內的固定單位（例如資料取用委員會）負責，而非由專案中的研究人員負責。
~~~
